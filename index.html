Learn Docker from Scratch

Docker is a tool that helps you package and run applications in isolated environments called containers, making development and deployment easier.

What is Docker?

Docker is an open-source platform that automates the deployment, scaling, and management of applications using containerization. Containers allow you to package an app with all its dependencies into a standardized unit.

Before Docker vs With Docker

Before Docker

  • Manual setup of environments
  • Dependency conflicts
  • Difficult to share and deploy
  • "Works on my machine" issues

With Docker

  • Consistent environments
  • Isolated dependencies
  • Easy sharing and deployment
  • Portable across systems

Key Terms

Image

A lightweight, standalone, executable package that includes everything needed to run a piece of software.

Container

A runnable instance of an image. You can start, stop, and delete containers.

Docker Engine

The core part of Docker that creates and runs containers using images.

Docker Hub

A cloud-based registry where you can find and share Docker images.

Installation Guide

Windows

  1. Download Docker Desktop from docker.com
  2. Run the installer
  3. Follow the setup wizard
  4. Start Docker Desktop

macOS

  1. Download Docker Desktop from docker.com
  2. Open the .dmg file
  3. Drag Docker to Applications
  4. Launch Docker Desktop

Linux

  1. Update your package index: sudo apt-get update
  2. Install Docker: sudo apt-get install docker.io
  3. Start Docker service: sudo systemctl start docker
  4. Enable Docker to start on boot: sudo systemctl enable docker

Basic Docker Commands

docker --version

Check Docker version

docker --version

docker pull

Download an image from Docker Hub

docker pull hello-world

docker images

List downloaded images

docker images

docker ps

List running containers

docker ps

docker run

Run a container from an image

docker run hello-world

docker stop

Stop a running container

docker stop <container_id>

docker rm

Remove a stopped container

docker rm <container_id>

Build & Run Your First Container

1. Pull an Image

Download the hello-world image from Docker Hub.

docker pull hello-world

2. Run the Container

Start a container from the image.

docker run hello-world

3. Check Running Containers

See if any containers are running.

docker ps -a

4. Stop and Remove

Stop and clean up the container.

docker stop <container_id> && docker rm <container_id>
